# 🔗 URL Shortener

A simple and responsive URL Shortener web application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://yugandharatech2025-parab.github.io/URLs-Shortner/

## 📸 Preview

![URL Shortener Preview](screenshot.png)

## ✨ Features

- Clean and user-friendly interface
- Generate short link IDs from long URLs
- URL input validation
- Responsive design
- Fast and easy to use
- Interactive user interface
- Original URL can be opened through the generated link

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Remix Icon
